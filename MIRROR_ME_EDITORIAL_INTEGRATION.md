# MIRROR ME — Intégration éditoriale chirurgicale

Branche de travail : `safe-css-migration`

Objectif : appliquer la ligne rédactionnelle validée sans écraser l’existant.

---

## Règle absolue

Ne jamais remplacer une page complète par une version simplifiée.

Ne jamais supprimer :
- la navigation ;
- le footer ;
- les images ;
- les scripts ;
- les IDs ;
- les classes existantes ;
- les blocs structurels.

L’intégration porte uniquement sur :
- H1 ;
- intros ;
- paragraphes éditoriaux ;
- textes de CTA ;
- meta descriptions.

La ligne éditoriale est volontairement architecturale, froide, tenue et précise.

Vouvoiement partout.

La plupart des personnes qui s’adressent à moi ont une excellente culture d’elles-mêmes. Elles ont lu, analysé, intellectualisé. Elles savent précisément pourquoi elles bloquent. Le piège actuel est là : confondre la compréhension d’un schéma avec sa résolution.

## Le passage à l’action

Mirror Me ne propose pas une énième exploration théorique du passé. Nous mobilisons vos ressources actuelles pour concevoir de nouveaux modes de réponse. Nous passons de la théorie à l’expérience terrain.

## Le basculement

Lorsque vous changez votre mode de réponse sur un territoire, l’environnement se réajuste immédiatement. Ce n’est pas de la magie, c’est de la mécanique relationnelle.
